Endpoints:

  • /data
    • GET /data => get current data (clients and networks)
    • GET /data/all => get current data (clients and networks)
    • GET /data/clients => get only clients
    • GET /data/networks => get only networks

    • POST /data => upload csv file with data


  • /test
    • GET /test => get random test data (clients and networks)
    • GET /test/all => get random test data (clients and networks)
    • GET /test/clients => get random test clients
    • GET /test/networks => get random test networks
